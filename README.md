## Portfolio Overview

The written portfolio addresses **Domain 2: Route Finding**, implementing a campus
emergency route finder using informed and uninformed search algorithms. Key topics covered:

- Graph representation of a constrained real-world environment
- A* Search with an admissible heuristic, compared against UCS and BFS
- Ethical analysis of rule-based vs. neural approaches to safety-critical AI

## Module Topics

- Search algorithms (BFS, DFS, UCS, A*)
- Natural language processing (rule-based sentiment classification)
- Reinforcement learning (Q-learning, reward shaping, hyperparameter tuning)
- AI planning (PDDL, blocks world, Pac-Man, robotic floor cleaning)
